# Configuration Repository pour Spring Cloud Config Server

Ce dépôt contient toutes les configurations externalisées pour les microservices.

## Structure

- `application.yml`: Configuration commune à tous les services
- `svc-auth.yml`: Configuration spécifique au service Auth
- `svc-menu-commandes.yml`: Configuration Menu & Commandes
- `svc-cuisine.yml`: Configuration Cuisine
- `svc-inventaire-prevision.yml`: Configuration Inventaire
- `svc-rh-planning.yml`: Configuration RH
- `svc-notifications-reporting.yml`: Configuration Notifications
